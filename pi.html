Main Focus:learning to use RasberryPi:

For the most part, RasberryPi amazed me. I had never really worked with the physical configuration and circuits of the actual computer software itself, and seeing that the low priced and very small RasberryPi was able to run a software compared to that of any computer was very impressive. To be able to run various programs on the RasberryPi you miust use either Python or Scratch, personally I really enjoyed running the programn that read my name, to see it transferred from the desktop was very impressive for me. I also really enjoyed playing MinecraftPi. My main takeaway, in this project was just hoe much can be accessed using a 2 by 2 circuit board, and learning to run programns.
